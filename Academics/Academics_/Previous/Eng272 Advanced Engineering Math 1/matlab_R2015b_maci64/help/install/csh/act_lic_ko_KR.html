라이선스 파일 경로 지정

인터넷에 연결하지 않고 활성화하려면 라이선스 파일이 있어야 합니다. 라이선스 파일이 설치를 진행하는 사용자가 실행할 수 있는 제품을 식별합니다. 라이선스 담당 관리자는 MathWorks 웹 사이트의 라이선스 센터에서 라이선스 파일을 가져올 수 있습니다. 라이선스 파일 경로 입력(Enter the path to the License File) 옵션을 선택하고 텍스트 상자에 라이선스 파일의 전체 경로를 입력한 후(또는 파일을 끌어서 놓은 후) 다음(Next)을 클릭합니다.

라이선스 파일이 없는 경우 라이선스 파일을 갖고 있지 않습니다(I do not have a license file) 옵션을 선택하고 다음(Next)을 클릭하여 라이선스 파일을 가져오는 방법을 확인합니다.
