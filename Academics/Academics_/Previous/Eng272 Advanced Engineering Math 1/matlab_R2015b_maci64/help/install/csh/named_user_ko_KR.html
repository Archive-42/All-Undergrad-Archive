사용자 이름 지정

명명된 독립 사용자(Standalone Named User) 활성화 유형을 선택한 경우 이 소프트웨어를 사용할 사용자의 운영 체제 사용자 이름을 지정해야 합니다. 명명된 독립 사용자(Standalone Named User) 활성화 유형은 이 소프트웨어의 사용을 특정 컴퓨터의 특정 사용자로 제한합니다. MathWorks®에서는 운영 체제 사용자 이름을 사용하여 이 사용자를 식별합니다. 운영 체제 사용자 이름은 사용자가 컴퓨터에 액세스할 때 사용하는 ID입니다. 이 ID는 컴퓨터 로그인 이름으로도 알려져 있습니다. MathWorks 소프트웨어를 사용하려면 설치를 진행하는 본인이 지정한 사용자 이름으로 컴퓨터에 로그인해야 합니다.

기본적으로, 활성화 응용 프로그램은 이 활성화 응용 프로그램을 실행하는 사용자를 사용자 이름에 입력합니다. 이 디폴트 값을 수락하려면 본인이 소프트웨어를 사용함(I will use the software) 옵션을 선택된 채로 두고 다음(Next)을 클릭합니다. 관리자 계정을 사용하여 소프트웨어를 설치했지만 이 소프트웨어에 액세스할 때는 다른 계정을 사용하려는 경우 여기에서 원하는 사용자 이름을 지정할 수 있습니다.

다른 사용자를 위해 라이선스를 활성화하려면 다른 사람이 이 소프트웨어를 사용함(Someone else will use the software) 옵션을 선택하고 해당 사용자의 전자 메일 주소, 이름, 운영 체제 사용자 이름을 지정한 후 다음(Next)을 클릭합니다. 활성화 응용 프로그램이 지정한 MathWorks 계정을 찾거나 새로 계정을 만듭니다. 이 옵션은 다른 사용자를 위해 소프트웨어를 활성화해야 하는 시스템 관리자에게 유용할 수 있습니다.
