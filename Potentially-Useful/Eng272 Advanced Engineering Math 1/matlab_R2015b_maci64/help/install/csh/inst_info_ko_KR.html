파일 설치 키 입력

인터넷에 연결되어 있지 않은 경우 수동 설치를 선택하면 파일 설치 키 대화 상자가 표시됩니다. 파일 설치 키로 설치할 수 있는 제품을 식별합니다.

파일 설치 키가 있는 경우 내 라이선스에 대한 파일 설치 키를 가지고 있습니다(I have the File Installation Key for my license) 옵션을 선택하고 이 키를 입력한 후 다음(Next)을 클릭합니다. 라이선스 담당 관리자는 MathWorks® 웹 사이트의 라이선스 센터에서 파일 설치 키를 가져올 수 있습니다.

파일 설치 키가 없는 경우 파일 설치 키를 가지고 있지 않습니다(I do not have the File Installation Key) 옵션을 선택하고 다음(Next)을 클릭합니다. 인스톨러에 키를 받는 데 필요한 정보가 표시됩니다.
