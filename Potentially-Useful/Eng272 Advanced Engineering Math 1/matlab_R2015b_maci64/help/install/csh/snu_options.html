Choose Authentication Options

  • If you want to activate MATLAB® for use on this computer, select Authorize this computer now.

    A License File that grants you the right to use MATLAB is stored on your computer.

  • If you prefer to use an internet connection to retrieve your license rights each time you start MATLAB, select Authenticate (log in) each time the software starts.

    If you select this option, your computer must have an uninterrupted internet connection to MathWorks® while you use the software. No License File is stored on your computer.
