Choose to Install or Download

Chose whether to install MATLAB® products immediately or download the software so you can install MATLAB products at a later time.

  • To install the software immediately, choose Install .

    Choosing this option allows you to download MATLAB products from MathWorks® and install the software on the computer where you are currently logged in.

  • To download the software now and install the MATLAB products at a later time, choose Download Only .

    Choosing this option allows you to download the software from MathWorks to a specified download folder and install the software at a convenient time. You can install the products for one or more computers on one or more platforms.
